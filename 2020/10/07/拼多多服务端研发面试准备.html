拼多多服务端研发面试准备

问答

  • 项目相关技术栈、中间件
  • 项目中有难度、有挑战的事情
  • 日常有没有参与经历过开源项目，看过什么源码，对该技术的理解
  • redis性能高效的原因(重新实现了数据结构、IO多路复用、多路复现的底层实现epoll，单线程基于内存)
  • 持久化方式rdb、aof
  • 学习了哪些语言、框架
  • SpringMVC工作原理
  • 让你写一个聊天软件，你该如何设计，数据如何传输、编解码
  • 对排序有什么比较深刻的了解吗
  • redis删除一个有百万数据的set要怎么做
  • 如何确保发送邮件成功并只发送一次？

Java

  • 异常体系设计，异常分类及处理
  • runtime异常必须要捕获吗
  • GC过程，调优过程、死锁处理
  • 线程池设计，线程数量如何配置选择（高低并发、任务执行时间长短的各种场景）
  • synchronized和lock的实现，synchronized底层实现使用方法和应用场景、锁升级
  • 简单介绍volatile，和synchronized的区别
  • 手写并发的HashMap
  • HashMap和TreeMap的底层实现和put的时间复杂度
  • JVM的内存区域
  • JVM的垃圾回收算法
  • 加密的算法
  • 类加载机制了解吗？1.8的类在1.7中能被正确加载吗？不能的话是在那一步判断的？这一步具体检查什么东西？
  • java常见的几种调用机制
  • 介绍一下回调机制
  • 消息队列

操作系统

  • 基础知识八股文，进程、线程的区别，线程同步、进程通信
  • 常用的linux命令
  • linux如何创建一个新文件
  • linux进程有哪些特点
  • 如果说用过ping、ssh，会问计网中，ping、ssh分别属于哪一层，实现方式
  • 多线程在操作系统内部如何实现

数据库

  • 索引的实现方式
  • hash、B+、B树实现的优劣对比（MySQL、MongoDB分别是怎么实现的）
  • 数据库的事务、隔离级别、实现方式
  • MySQL中实现分页功能的语句
  • MySQL索引的对比，各自底层数据结构
  • 辅助索引的实现机制，与聚簇索引的关系，具体存储的是什么
  • MySQL事务隔离的实现原理，锁有哪些？日志有哪些？

计网

  • 三次握手、四次挥手
    • 为什么握手要3次，两次可以吗？四次可以吗？
    • 如果server端没有收到第三次ack，但是收到了client端发送的数据，server端会怎么处理
    • 为什么挥手需要四次
  • 介绍一下tcp，如何保证可靠传输
  • http 1/1.1/2的区别
  • https相关问题
  • 打开一个网站会发生哪些事情
  • 网络传输序列化和大小端的关系
